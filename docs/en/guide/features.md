## Shift Types

| Code | Name | Hours |
|---|---|---|
| **D** | Duty | 7:00–19:00 (12h) |
| **D\*** | Alternative Duty | 8:00–20:00 (12h) |
| **R** | Morning | 7:00–14:35 (7:35h) |
| **•** | On-call | 0h |

Custom time ranges are also supported.
